    Node Weekly #428

    2022/3/11

    (opens new window)

    What's Really Going On Inside Your node_modules Folder? (opens new window) — A running joke is that node_modules folders are so huge they’re heavier than black holes (opens new window) but when packages get taken over by nefarious groups, the contents of node_modules becomes somewhat less funny. This post looks into what malicious packages can do and the broad problem of supply chain attacks generally.

    Node v17.7.0 (Current) Released (opens new window) — Updates to nghttp2 and npm (8.5.2), some new options (opens new window) for net.Socket and net.Server, and Ben Noordhuis (one of the most prolific Node contributors who stepped back (opens new window) as a core committer for reasons (opens new window) in 2013) is officially fully back as a Node.js collaborator (though he has continued (opens new window) to contribute code all along).

    (opens new window)

    Introducing the Elastic CI Stack for EC2 Mac (opens new window) — This new open source stack is created specifically for mobile teams, helping you migrate to AWS-managed servers for increased reliability, security, and speed. 📱

    Socket: See Potential Security Issues for npm Packages (opens new window) — An interesting new project that scans the code of each npm package in an attempt to characterize their behavior which is then reported on project specific pages, such as this one for lodash (opens new window) or this one for zx (opens new window).

    A Case Study on Moving from Next.js to Remix (opens new window) — Remix (opens new window) is the newest full stack Web framework on the block, and the first case studies are beginning to come in. In this case, the author covers the rewrite of his personal site (where this article is hosted).

    Making a Discord Playlist Bot with Serverless Cloud (opens new window) — Serverless Cloud is a serverless platform from Serverless Inc., the folks behind Serverless Framework (opens new window). (Have we said ‘serverless’ enough yet?) This post ties together Node.js with some specific Serverless Cloud features to easily create a Discord chat bot that can add songs to a shared Spotify playlist.

    A Guide to Node Process Management with PM2 (opens new window) — PM2 (opens new window) is one of the longest standing Node utilities used for managing processes and is worth checking out if you have a Node process you need to stay up 24/7.

    Malicious Node.js Packages: Niche Configurations & Invisible Characters (opens new window)

    Diving into Node's Streams (opens new window) — Streams provide a defined interface and abstraction over the idea of working with streaming data in Node – they seem to suffer from often being misunderstood, though, so tutorials always tend to be popular.

    ▶  USB Reverse Engineering and Writing Drivers (opens new window) — If you’ve got a hankering to do some hardware hackery with Node, you might enjoy this truly low level stream.

    ▶  Discussing Securing the Open Source Supply Chain with Feross Aboukhadijeh (opens new window) — Feross Aboukhadijeh is one of the minds behind Socket (featured above) and he joined the popular Changelog podcast to discuss the launch and why making the assumption that all your dependencies are malicious may be a necessary step to take.

    Deploying Your Node API to AWS Using Elastic Beanstalk (opens new window)

    The Fetch API is Finally Coming to Node (opens new window)

    # 🛠 Code & Tools

    (opens new window)

    PSD: A Zero-Dependency PSD (Photoshop) Parser for Browser and Node.js (opens new window) — Will parse info for each layer including text and also supports Photoshop’s .psb (big image) format. GitHub repo (opens new window).

    Undici 4.15: The Fresh HTTP/1.1 Client for Node (opens new window) — Undici’s goal to be the best HTTP/1.1 client for Node takes another step forward.

    Bree 8.0: A Versatile Job Scheduler for Node (opens new window) — Supports cron, dates, ms, later, and human-friendly for scheduling things.

    Stream Video in Your Node App in Two API Calls (opens new window)

    exiftool-vendored: Fast, Cross-Platform Node.js Access to ExifTool (opens new window) — Use this when you want to access embedded EXIF data within image files (particularly those taken with phones or DSLRs).

    elasticsearch-js 8.1.0: Official Elasticsearch Client for Node (opens new window) — Elasticsearch (opens new window) is a great open source search database system for adding powerful search features to your apps. This update adds Elasticsearch 8.1 compatibility.

    AVA 4.1: The Node.js Test Runner (opens new window) — A popular test runner known for its simplicity and speed.

    fastify-websocket: Basic WebSocket Support for Fastify (opens new window)

    Dynamodump 2.0: A CLI Tool for Backing Up and Restoring Schema and Data from DynamoDB (opens new window)
