    深入了解 React Native 的新架构 (opens new window) — React Native 的新架构将在 2022 年推出，正如 React Native 团队的这篇 帖子 (opens new window) 中所描述的。这里有一些独立的观察。

    一些不喜欢 React Hooks 的原因? (opens new window) — 令人惊讶的是，截止这个月，hooks 已经陪伴我们三年了!(从 React 16.8 算) 它们现在是 React 领域不可或缺的一部分。然而，对于它们的潜在弱点，总是有一点反省的余地。

    (opens new window)

    真相就在代码中：用 Sourcegraph 修复漏洞 (opens new window) — 你可以确信你找到了每一个受影响的代码实例，并通过自动修复和拉动请求跟踪你的修复措施的完成情况，从而补救漏洞。有了 Sourcegraph，通过搜索找到及时的解决方案。

    从 React Native 调用 Windows APIs 变得更容易 (opens new window) — 刚刚宣布的 React Native WinRT (opens new window) 项目将使开发人员能够直接从 JavaScript 调用 Windows Runtime APIs。那么你能用它做什么呢？比如说 Windows 11 Settings page (opens new window)。

    🎧  The React Show: 一个 React 播客 (opens new window) — 这个播客现在已经有 37 集了，已经涵盖了各种各样的 React 主题。大致60-90分钟的形式提供了大量的机会来深入报道一些话题，最新一集 (opens new window) 涵盖了 React’s Fiber (opens new window) 的调用和机制。

    2022 React 新手指南 (opens new window) — 我们已经出版了 276 期，有超过 50,000名 订阅者。这本指南提供了基础知识，让你开始行动。

    ❓  如果 Meta/Facebook '挂掉', React 会一起挂掉吗? (opens new window) — 一个有趣的、开放式的讨论，至少从这个相当确定的问题开始，但很快就演变成一个相当高的讨论，讨论 Meta 未来的各种潜在问题，以及这些问题如何影响（或不影响）React 的未来。

    # 🛠 代码和工具

    (opens new window)

    React95 3.11: Windows 95 风格的 React 组件 UI 库 (opens new window) — 一组有趣的以 Windows 95 为主题的 UI 组件，用于 React 应用程序，并使用 styled-components 构建。享受他们的 非常复古的主页 (opens new window) 当然，我们必须庆祝这个 3.11 版本，向 Windows 3.11 致敬!

    Griffel: 预编译的 CSS-in-JS (opens new window) — 来自微软的另一个 CSS-in-JS 解决方案的选择。这个方案拥有近乎零的运行时间，支持 SSR，以及用 JS 对象定义的样式，以及其他功能。

    Slinky: 在 Scala 中写 React 应用 (opens new window) — 在 React 中利用灵活的面向JVM的 Scala (opens new window) 语言的面向对象和功能性编程能力。

    为何说把内容迁移到无头内容管理系统是你最后一次迁移 (opens new window)

    react-tree: 一个用于 React 的分层对象树组件 (opens new window) — demo 在这 (opens new window)。支持深色、浅色和自定义主题，以及多种宽度。

    React Stripe.js: 为 Stripe.js 和 Stripe 元素构建的 React 组件 (opens new window)

    react-native-progress: 基于 SVG 的 React Native 进度指示器和旋转器 (opens new window)

    react-search-autocomplete (opens new window) — 可定制的搜索框，具有基于模糊搜索和过滤结果的自动完成功能。

    react-canvas-confetti (opens new window) — （掘金投票的特效）没能在超级碗之前上线..

    easytimer-react-hook (opens new window) — 配置一个秒表或倒数计时器。

    react-custom-flag-select (opens new window) — 在一个选择控件中显示一个国旗以及描述（fun with flags）。
