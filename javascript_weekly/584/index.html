    JavaScript Weekly #584

    2022/4/9

    (opens new window)

    Veni, vidi, formatae! Announcing Rome Formatter: Super Fast JavaScript Formatting (opens new window) — Rome (opens new window) is an ambitious project that sets out to replace a lot of JS tools in one hit. But Rome wasn’t built in a day, (opens new window) so they’re unveiling their take on Prettier-esque code formatting first. You can use it from the CLI but they’re recommending this VS Code extension (opens new window) primarily as a way to get a taste.

    Rome Team

    Writing Redux Reducers in Rust (opens new window) — We often cover how Rust is worming its way into numerous JavaScript areas (e.g. Rome, above!) but this is an interesting one: using Rust to write complex functions that are compiled down to WASM and used in a React/Redux app.

    Arend van Beelen

    (opens new window)

    Wallaby.js: 15x Faster JavaScript/jest Testing and Debugging (opens new window) — Wallaby runs your tests immediately as you type, with runtime values, errors and code coverage displayed in real-time right next to your code. Features have been designed specifically for testing, including new smart start (opens new window) which only runs test files you are currently working on.

    Wallaby.js sponsor

    JavaScript & Node Testing Best Practices: 2022 Edition (opens new window) — Fifty best practices divided into categories (testing, backend, frontend, etc.) complete with simple examples. We first linked to this 2½ years ago, but a new edition is now out, updated for 2022 standards and with translations in seven other languages including Chinese, Spanish and French.

    Yoni Goldberg

    Announcing RedwoodJS 1.0 and $1M Funding (opens new window) — Tom, formerly a co-founder of GitHub, announces the 1.0 release of RedwoodJS (opens new window), a full-stack framework perhaps originally best associated with the Jamstack (opens new window) but is essentially a React and GraphQL-powered framework for building whatever apps you’d like.

    Tom Preston-Werner

    🇺🇦 Devs For Ukraine: An Online Event to Support Ukraine (opens new window) — Well known names including Addy Osmani, Cassidy Williams, Dan Abramov, and Sandi Metz are coming together on April 25-26 with some great (online) talks to help raise funds for Ukraine-related causes.

    Remote

    # IN BRIEF:

    • The British government has removed jQuery as a dependency from its gov.uk (opens new window) frontend apps – Matt Hobbs shows off how this improves performance (opens new window) significantly.

    • If you use AWS Lambda for running serverless functions, you can now get direct HTTPS URLs (opens new window) for triggering functions, rather than having to use API Gateway. Serverless Framework already supports them (opens new window) too. (→ via Serverless Status (opens new window))

    • Node 12.22.12 (LTS) (opens new window) is out and it's the last Node 12 ever 👋 Node 17.9.0 (Current) (opens new window) is also out, but is a minor release.

    • Almost a third of the top 35 npm packages remain at risk (opens new window) due to a lack of 2FA use by associated accounts.

    • A developer makes the argument that JavaScript is safer for programming with integers than modern C (opens new window) – it's no April Fool.
